# 校付通 × 微企付 · 缴费支付交互 Demo

教育行业 SaaS（校付通）接入腾讯微企付的缴费支付全流程交互原型，单文件 HTML，零依赖。

## 在线预览

打开 GitHub Pages：https://zpbxxyy.github.io/xft-wqf-edu-demo/

## 流程概览

10 屏一镜到底，全可点击：

1. 我的校付通 · 信息核对
2. 微信会话 · 校付通通知
3. 校付通 · 待办通知
4. 缴费详情 · 选费目
5. 订单确认 · 学生身份
6. 校付通收银台 · 默认勾选「腾讯微企付（推荐）」+ 红包标签 + 折叠隐藏微信支付
7. 腾讯微企付 · 选卡（招商 / 工商 / 中国 / 建设）
8. 银行 · 安全支付确认页（按所选银行联动）
9. 银行 · 输入支付密码（自动填入动画）
10. 支付成功

## 本地查看

```
git clone https://github.com/zpbxxyy/xft-wqf-edu-demo.git
cd xft-wqf-edu-demo
open index.html
```

单文件可直接双击浏览器打开。
